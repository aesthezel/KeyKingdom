# KeyKingdom
Proyecto clave para entender el sistema de modding dentro de videojuegos hechos en Unity, aunque, el objetivo final es extenderlo a otros motores y librerias basadas en C#.

## Documentación
Para detalles sobre la arquitectura, patrones utilizados y cómo crear nuevos mods, consulta la [Guía de Modding](MODDING_GUIDE.md).
